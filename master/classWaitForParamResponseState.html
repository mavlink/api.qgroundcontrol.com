Detailed Description

Waits for either PARAM_VALUE (success) or PARAM_ERROR (rejection) from the vehicle. On PARAM_VALUE matching predicate: calls waitComplete() (emits advance()) On PARAM_ERROR matching predicate: stores error info, calls waitFailed() (emits error()) On timeout: existing timeout() signal fires (for retry path)
